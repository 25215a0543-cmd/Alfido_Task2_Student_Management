# 🎓 Student Management System

## 📌 Project Overview

The **Student Management System** is a console-based Java application developed as part of the **Alfido Tech Java Developer Internship - Task 2**.

This project demonstrates Java fundamentals by implementing a menu-driven application that allows users to manage student records. It uses arrays for data storage and provides various operations such as adding, viewing, searching, updating, and deleting student records.

---

## 🎯 Objective

The objective of this project is to understand and apply Java programming fundamentals, including:

- User input using Scanner
- Conditional statements
- Loops
- Methods
- Arrays
- Switch-case
- Console-based application development

---

## ✨ Features

- Add Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student Record
- Calculate Average Marks
- Display Student with Highest Marks
- Display Student with Lowest Marks
- Count Total Students
- Exit Program
